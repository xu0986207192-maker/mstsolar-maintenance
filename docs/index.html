🔢 PR 值與 RA 值計算器

📊 PR 值（Performance Ratio）計算

PR 值 = 實際發電量 ÷ (日照強度 × 系統容量 × 時數) × 100%

計算結果：
評價標準：
優良：PR ≥ 80% | 良好：70% ≤ PR < 80% | 普通：60% ≤ PR < 70% | 需改善：PR < 60%

⚡ RA 值（太陽能系統可用率）計算

RA 值 = (實測串列功率 ÷ 額定裝置容量) × 100%

於特定日照與溫度下量測的實際功率
標準測試條件(STC)下的額定功率
計算結果：
評價標準：
優良：RA ≥ 90% | 良好：80% ≤ RA < 90% | 普通：70% ≤ RA < 80% | 需改善：RA < 70%
